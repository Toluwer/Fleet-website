Install in a minute

From download to ready.

  1. 1
    Run FleetInstaller.exe

    Open the downloaded installer and choose Install. Fleet installs per user; administrator setup is not required by the installer configuration.

  2. 2
    Handle the Windows warning

    The current build is not code-signed. If SmartScreen appears, inspect the publisher information and file hash before choosing to continue.

  3. 3
    Confirm Roblox detection

    Fleet opens after setup. Instances should show a green Roblox detected banner and Multi-instance: ready.

  4. 4
    Add accounts or stay signed out

    Account sign-in is optional. Both launch modes support multiple isolated clients.

What the installer contains

No mystery helper.

The installer packages the Electron desktop app, its renderer, the koffi native runtime used for Win32 access, and update metadata. Fleet does not install a service or browser extension.

Desktop applicationFleet.exe and its Electron runtime.

Native access layerkoffi bindings for Windows process and window APIs.

Update supportNSIS metadata for reachable GitHub Releases.
