1.5.1LatestJuly 3, 2026

See who is in the server.

A dedicated People → Server workflow brings a bounded, local roster inspection into Fleet.

  • Added a Server subtab with roster search and two-second refresh while visible.
  • Shows detected username, display name, avatar, user ID, team, health, and max health.
  • Runs process-memory inspection in a worker with strict time, read-count, and child-count budgets.
  • Added the dedicated IPC/preload path and coverage for the memory traversal.
  • Improved portable-build diagnostics when Windows metadata tooling fails.

1.5.0July 2, 2026

Playtime, account context, cleaner cards.

Fleet became more informative without sending activity outside the PC.

  • Added local playtime analytics with today, seven-day, all-time, and active-session views.
  • Added per-account Robux and Premium status.
  • Improved game-card action wrapping to prevent clipping at narrower widths.
1.4.1July 2, 2026

Less setup friction.

A practical workflow release focused on getting into the right session faster.

  • Improved guidance for installs without saved accounts.
  • Added keep-alive automatic rejoin with cooldown and strike-out behavior.
  • Added favorite games, recent joins, and clipboard quick-join for copied Roblox links.
  • Improved login errors and the account sign-in user agent.
  • Polished custom installer centering and spacing.
1.4.0July 1, 2026

Server Intelligence arrives.

Public server selection became a real decision tool instead of a single API page.

  • Fixed Most Players by scanning past full Roblox server pages.
  • Added bounded deep scans and occupancy, ping, FPS, and free-slot filters.
  • Added quality scores, analytics, server ID copy, and optional live refresh.
  • Sanitized updater failures so raw transport headers and cookies never render.
  • Reworked the installer into a custom centered card with DPI-safe text.
