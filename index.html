Straight answers

Before you install.

Fleet is deliberately narrow: a Windows tool for managing Roblox clients you control.

Does Fleet modify Roblox?+

No. Fleet launches the installed Roblox player through lightweight directory junctions and manages Windows kernel objects. It does not patch Roblox files.

Where are account sessions stored?+

On the current PC, encrypted through Electron's safeStorage API, which uses Windows DPAPI. The website demo does not accept or store account data.

Can I launch without adding an account?+

Yes. Choose Signed out on the Instances page and select how many clients to open.

What happens when Roblox updates?+

Fleet detects the current Roblox installation path automatically. A future change to Roblox's single-instance mechanism could require a Fleet update; Diagnostics exposes the information needed to investigate it.

Which systems are supported?+

Fleet is Windows-only by design and supports 64-bit Windows 10 and Windows 11 with Roblox installed.
